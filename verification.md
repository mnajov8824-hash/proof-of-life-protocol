# Verification System (MVP)

## Purpose
Ensure that payments are made only for living animals.

## Proof of Life Requirements
- Monthly video submission
- Animal must be clearly visible
- Current date must be spoken or written in the video
- Video length: 10–30 seconds

## Submission Method
- Video uploaded via simple web interface (MVP)
- Each animal has a unique ID

## Verification Process
- Manual verification by independent reviewers
- Reviewers confirm:
  - Animal identity
  - Animal is alive
  - Date is current

## Fraud Prevention
- Random additional checks
- Permanent payment stop if fraud is detected

## Scalability Plan
- Manual verification → community verification → automated AI checks
